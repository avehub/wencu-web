# WenCuWeb

文促WEB端代码仓库

## 文促后端

 - https://github.com/avehub/wencu

## 二次开发

Node.js 版本 14.16.0 或以上 (可用 nvm 管理)

**前端初始化**

注意修改 `nuxt.config.js` 中的 `proxy` 配置，指向后端地址

```bash
# 安装依赖
npm install

# 运行前端，监听3000端口，浏览器访问 http://localhost:3000
npm run dev
```

## 分支说明

- master    魔豆文库社区版前端社区版主分支
- dev       魔豆文库社区版前端开发分支
- pro       魔豆文库专业版前端分支
